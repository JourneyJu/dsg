# 任务接收模块
接收同步或异步任务，存入数据库，并解析任务后调用对应的项目执行模块进行执行
# 项目执行模块
预置执行规则及对应的处理SQL，提交虚拟化引擎进行执行
# 探查报告模块
保存探查报告，提供探查报告查询，html，pdf类型报告生成等服务
# 智能分析模块
提供语义化探查，将用于探查要求转换为标准SQL执行
提供探查结果理解，将用户探查的数据结果进行语义化分析理解输出
# 文件服务模块
提供输入输出的文件保存，下载等功能
# 消息处理模块
提供异步任务接收和任务状态执行通知等功能
